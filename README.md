## Prereqs
- NPM installed
- Update 'API_BASE_URL' in 'src/constants.js' with the base url for the backend (e.g. http://127.0.0.1:5000)

## Starting app
Run the commands below in the root directory
```bash
npm run install
npm run dev
```
