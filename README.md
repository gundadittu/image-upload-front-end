## Prereqs
- Install Node.js and Npm: https://nodejs.org/en/download
- Update ```API_BASE_URL``` in ```src/constants.js``` with the base url for the backend (e.g. http://127.0.0.1:5000)

## Start React app
Run the command below in the root directory
```bash
npm run dev
```
